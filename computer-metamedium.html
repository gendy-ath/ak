Computer Metamedium
Are the symbols, images and data int the world wide web delimited and defined in a default topology? Could these addresses be perceptible in the physical space? Through the use of especially customised algorithm the computer receives data in real-time from “Twitter’ and in particular form a predefined hashtag address. At the same time these incoming data were being captured and stored in a timeline. Moreover the algorithm extracts all the symbols included the incoming tweets in an 8-bit representation. These extracted values are the raw material for the processing of a new generating structure that produces a singular visualized approach of digital space each time. Thus for every new data packet a new visualised structure is being generated. The whole processing depends on the specific time moment that the incoming data are being captured, in order to achieve a time-based scaling.
